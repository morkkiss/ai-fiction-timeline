# Faculty Accessibility Toolkit

Plain-language WCAG 2.1 AA accessibility resources for university faculty.

Prepared by Morgan Talty, Department of English, University of Maine, April 2026.

Live site (after enabling GitHub Pages with /docs source): https://morkkiss.github.io/ai-fiction-timeline/accessibility/

## Pieces

- `index.html` — Front door (formerly _START_HERE)
- `Faculty_Field_Guide_2026-04-30.html` — Polished docs app with full-text search, tool filter chips (Word / Google Docs / PowerPoint / PDF / Brightspace / Video / Web), 28 sections, glossary tooltips, mobile responsive
- `Faculty_WCAG_Checker_2026-04-30.html` — Interactive WCAG checker with plain-language findings
- `Faculty_WCAG_Routing_Guide_2026-04-30.html` — Routing matrix
- `Faculty_Examples_2026-04-30.html` — Side-by-side before/after demos: Course Page, Equations (MathML), Forms
- `Faculty_Field_Guide_2026-04-30.md` — Markdown export
