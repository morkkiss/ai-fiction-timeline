A decision tree for faculty questions

Where to go when the checker or guide can't answer

UMaine's official Title II page sets the rule, the deadline, and the baseline campus tools. This page does the next job: it routes unresolved questions to the right source instead of leaving you with a generic "get help" message.

Use this page when the question isn't simply "fix alt text" or "caption the video," but "whose problem is this," "which office owns it," or "which source should I trust next?"

Routing matrix

Question
Where to go
What does the law or the deadline actually require?
UMaine Title II page first. If you need the federal text, the DOJ Title II rule is the source.
How do I do this in Brightspace, Kaltura, Google Docs, or Zoom?
CITL plus the vendor's own docs. Use vendor docs for exact clicks, settings, and limits.
A student has an accommodation letter or needs an exception.
Student Accessibility Services first; ADA Coordinator if you need formal institutional guidance. Don't try to answer this from a checklist.
This seems like an IT or vendor problem, not a faculty content problem.
UMS IT accessibility standards, CITL, and the platform's accessibility docs or VPAT.
This is a workload, impact, or responsibility dispute.
AFUM for labor and workload issues. Document what's being asked of you and what support you've been provided.
I understand the issue but still need the easiest fix path.
Field Guide and Checker (in this packet). Start there; come back here only if those aren't enough.

Authoritative source stack

  • Federal rule: DOJ Title II web rule for baseline legal scope and deadline.
  • Institutional baseline: UMaine Title II page and UMS accessibility policy.
  • Campus operations: CITL, Student Accessibility Services, ADA Coordinator, AFUM.
  • Vendor truth: Anthology Ally docs, Google accessibility help, Kaltura caption docs, Zoom transcript docs, vendor VPATs when available.
  • This toolkit: Field Guide, Checker, Routing Guide.
