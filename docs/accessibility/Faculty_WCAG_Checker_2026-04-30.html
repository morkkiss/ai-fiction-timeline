What this is: a fast accessibility check for an HTML version of a course page. Paste your content below, run an audit, and read the plain-language findings. Nothing leaves your browser — the audit runs locally on your machine.

Where do I get HTML to paste here?
  • Brightspace page: open the page editor → click the source-view button (often labeled </> or "View Source") → copy everything.
  • Word document: File → Save As → "Web Page" → open the resulting .html file in TextEdit → copy.
  • Google Doc: File → Download → Web Page (HTML, zipped) → unzip, open the .html file in TextEdit → copy.
  • Course website: in your browser, right-click the page → View Page Source → copy.

If "HTML" is unfamiliar, use the Field Guide instead — it walks through Word, Google Docs, and PowerPoint in plain language.
